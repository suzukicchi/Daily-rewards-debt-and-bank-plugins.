# Feaber MC - 統合経済システム

このプラグインは、Feaber MC サーバー専用の統合経済管理システムです。ログインボーナス、ローンシステム、および銀行口座管理機能を１つのプラグインで提供し、サーバー内の経済活動をサポートします。

## 概要
当サーバーの経済を活性化させるための中心的なシステムです。プレイヤーは日々のログインや預金運用を通じて資産を形成し、必要な資金をローンで借り入れることができます。

## 主な機能
- **ログインボーナス管理**: ログイン報酬を管理するGUIシステム。[cite: 1]
- **ローンシステム**: 資金の借入と返済を行う融資機能。[cite: 1]
- **中央銀行**: 現金を預け、利息を運用できる銀行システム。[cite: 1]

## コマンドリファレンス
| コマンド | 説明 |
| :--- | :--- |
| /loginbonus (または /daily) | ログインボーナスGUIを開きます。[cite: 1] |
| /loan (または /ローン) | ローン窓口GUIを開きます。[cite: 1] |
| /bank (または /銀行) | 銀行口座GUIを開きます。[cite: 1] |

## 管理者向けコマンド
権限を持つユーザー（OP含む）は、以下のコマンドで各種データの調整が可能です。

### 銀行口座調整
`/bank <set|add|remove> <金額> <プレイヤー名/@a/@r/@p/@s>`

### ローン調整
`/daily loan <set|add|remove> <金額> <プレイヤー名/@a/@r/@p/@s>`

### 操作解説
- **set**: 指定した金額に強制変更します。
- **add**: 指定した金額を加算します。
- **remove**: 指定した金額を減額します。

### 情報取得
- `/daily info` : 自分の情報を確認します。
- `/daily info <プレイヤー名>` : 指定したプレイヤーの情報を確認します。
※他プレイヤーの情報閲覧には `loginbonus.info` 権限またはOP権限が必要です。

## 設定ファイル (config.yml)
サーバー運用に合わせて以下の項目をカスタマイズ可能です。

### ログインボーナス設定 (login-bonus)
- **base-reward**: 1日あたりの基本支給額。[cite: 4]
- **reset-on-miss**: ログインを逃した場合に連続カウントをリセットするかどうか。[cite: 4]
- **monthly-streaks**: 月ごとの連続ログイン日数に応じた追加ボーナス報酬。[cite: 4]

### ローンシステム設定 (loan)
- **max-limit**: ローン借入の最大限界額。[cite: 4]
- **unit**: GUIボタン1回操作あたりの借入・返済単位。[cite: 4]
- **block-bonus-threshold**: ログインボーナスが差し押さえられる借入額の境界線。[cite: 4]
- **interest-rate**: 日ごとの借入利息（金利）。[cite: 4]
- **due-days**: 全額返済までの猶予日数。[cite: 4]

### 銀行システム設定 (bank)
- **max-limit**: 預金可能な最大金額（デフォルト: 1,000万）。[cite: 4]
- **interest-rate**: 定期利息の金利（デフォルト: 2%）。[cite: 4]
- **interest-interval-hours**: 利息が付与される時間間隔（デフォルト: 24時間）。[cite: 4]
